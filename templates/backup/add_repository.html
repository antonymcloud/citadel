Encryption

When initializing a new repository, choose an encryption mode:

  • None: No encryption (not recommended)
  • repokey: Encryption key stored in the repository
  • repokey-blake2: Like repokey but with stronger hashing
  • keyfile: Encryption key stored separately from repository
